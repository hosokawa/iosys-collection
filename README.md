# IOSYS Collection Helper

Chrome extension scaffold for quick IOSYS search shortcuts.

## Scope

- Enabled only on `iosys.co.jp` and its subdomains.
- Adds a right-click menu group named `IOSYS`.
- Provides submenu items for:
  - `Pixel 8(GZPFO)`
  - `Pixel 9(G1B60)`
  - `Pixel 10(GL066)`

The click actions are intentionally left as placeholders for the next step.
